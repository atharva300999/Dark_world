# 🔥 Raven TG ID to Number Bot

A powerful Telegram bot that converts Telegram User IDs to phone numbers with points system!

## ✨ Features

- 🔐 Force join channels verification
- ⭐ Points based system
- 🎁 Gift code generator
- 🫂 Referral program
- 📊 Admin panel
- 🚗 Vehicle number lookup
- 📜 GST details lookup
- 🏦 IFSC code lookup

## 🛠️ Commands

### User Commands
- `/start` - Start the bot
- `/services` - Show all services
- `/balance` - Check points balance
- `/refer` - Get referral link
- `/redeem` - Redeem gift code
- `/help` - Show help

### Admin Commands
- `/createcode <points> <max_uses> <expires_days>` - Create gift code
- `/addpoints <user_id> <points>` - Add points to user
- `/broadcast <message>` - Send broadcast message
- `/stats` - View bot statistics
